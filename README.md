# 자바 소스 코드

1. 클래스 알아보기
+ 생성자 : `class`를 객체로 생성하여 만들기 위한 대상
+ 변수 : 해당 자료형의 값을 담기 위하여 만들고 사용하는 대상
+ 상수 : 변수와 동일하지만 `한번 값을 넣고 수정을 할 수 없다는 특징`을 가지고 있습니다.
+ 메소드 : `기능`을 정의 하여 필요한 로직을 구현하는 대상
* `java` 파일를 컴파일 하기
```
javac Study.java
```
* `class` 파일를 실행 하기
```
java Study
```
